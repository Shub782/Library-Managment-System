**Library Management System**
A full-stack Library Management System built using Node.js, Express.js, EJS, and SQLite. This application is designed to help librarians manage books, students, borrowing records, and library visits through a simple and efficient interface.

**Features**
**Authentication**
Secure librarian login system
Session-based authentication
Logout functionality

**Book Management**
Add new books
Edit book details
Delete books
Track available quantity

**Student Management**
Add student records
Edit student information
Delete students
Search students by USN or name

**Borrowed Books**
Issue books to students
Return borrowed books
Automatic stock updates
Borrow status tracking

**Library Visits**
Record student entry and exit times
Calculate visit duration automatically
Search visits by USN

**Dashboard**
Total number of books
Total borrowed books
Total library visits

**Technologies Used**
Frontend: HTML, CSS, EJS
Backend: Node.js, Express.js
Database: SQLite3
Authentication: Express Session
Deployment: Render
Version Control: Git & GitHub

**Workflow**
1.Librarian logs into the system
2.Manages books and student records
3.Issues and returns books
4.Tracks library visits
5.Views system statistics on the dashboard
